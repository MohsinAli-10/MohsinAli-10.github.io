About Me

I'm Mohsin

an ML Engineer

Based in Lahore, Pakistan

Hi! I’m an AI and Machine Learning specialist with over 7 years of experience building innovative data-driven solutions. My passion lies in leveraging the power of machine learning to solve complex problems across industries, from digital signal processing to real-time web applications.

I began my journey at LUMS, where I honed my expertise in machine learning through hands-on projects like crowd density estimation, edge detection for image dehazing, and anomaly detection in EMG signals. Since then, I’ve pursued a Master’s degree focusing on advanced machine learning, deep learning, and linear algebra, further solidifying my technical foundation.

Over the years, I’ve completed more than 40 AI-focused projects, including the deployment of an end-to-end web application that predicts student performance using regression models. This project showcased my ability to build production-ready machine learning systems, integrating models with front-end applications using Flask, and deploying them on cloud platforms like AWS to ensure scalability and robust performance.

What I Do:

Data Science & ML Development: I specialize in data exploration, feature engineering, and selecting the right machine learning algorithms to extract meaningful insights from raw data. My experience spans from predictive modeling to anomaly detection and image processing.

Model Deployment & MLOps: I’m skilled in deploying machine learning models at scale, leveraging cloud infrastructure like AWS for real-world applications. From setting up Linux-based servers using Gunicorn to establishing CI/CD pipelines for streamlined deployments, I ensure that models not only work in development but thrive in production.

Future Focus: I’m currently expanding my expertise in machine learning engineering, particularly in model optimization techniques like quantization, pruning, and inference speed-ups. I'm exploring advanced MLOps tools and cloud-native architectures to handle the challenges of large-scale machine learning systems.

Whether it's deploying machine learning solutions in production, designing scalable infrastructure, or fine-tuning models for optimal performance, I’m dedicated to creating intelligent systems that make an impact.
